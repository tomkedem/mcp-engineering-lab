# Introduction

This repository accompanies the book "MCP Systems Engineering".

## What We Are Building

Throughout the book we build a single evolving system: mcp-engineering-lab.
It starts small and simple. By the end of the book it has structure,
tools, resources, permissions, logs, design patterns, and hardening.

## How to Use This Repository

Read the chapter first, then open the corresponding code.
Each chapter adds one layer. Do not skip ahead.

## Project Map

- docs/       — documentation per chapter
- design/     — architecture and design documents  
- server/     — the MCP server, evolving chapter by chapter
- samples/    — code samples
- tests/      — automated tests
